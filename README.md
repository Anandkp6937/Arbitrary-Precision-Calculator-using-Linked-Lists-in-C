# Arbitrary-Precision-Calculator-using-Linked-Lists-in-C

Built a high-precision arithmetic calculator in C using linked lists to perform operations on large numbers. Implements digit-wise addition, subtraction, multiplication, and division, overcoming limitations of standard data types.

usage:
gcc *.c -o apc
./apc {num1} {operation} {num2}

eg:

./apc 123 + 1
